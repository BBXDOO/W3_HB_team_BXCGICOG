# 🎯 Quick Start: ตั้งค่า GitHub Pages (3 ขั้นตอน)

## ขั้นตอนที่ 1: เปิด Settings
ไปที่: https://github.com/BBXDOO/W3_HB_team_BXCGICOG/settings/pages

## ขั้นตอนที่ 2: เลือก Branch และ Folder

```
Source:  Deploy from a branch  [dropdown ↓]
Branch:  <your-branch> (เช่น main)  [dropdown ↓]
Folder:  /docs  [dropdown ↓]
```

คลิก **[Save]**

## ขั้นตอนที่ 3: รอและทดสอบ

รอ 1-2 นาที แล้วเปิด:
```
https://bbxdoo.github.io/W3_HB_team_BXCGICOG/
```
