# Hybrid Model — Vision

เอกสารวิสัยทัศน์เชิงโครงสร้างของระบบ W3 (ระดับ abstract)
---
คำถาม : การตรวจสอบแท้ที่จริงคืออะไร ?
คำตอบ : คือความ "ไม่ใว้วางใจ".
คำถาม : แล้ว.. ระหว่างการตรวจสอบ กับ ไร้การตรวจสอบนั้น ..แบบไหนที่สำคัญ?
คำตอบ : ไม่ใข่แบบไหน ..แต่ สาระาำคัญคือ "มันใช้กับอะไร"?.
อย่าปล่อยให้สิ่งที่ซับซ้อนเข้ามาซ้อนความยุ่งเหยิงในชีวิตหรือกิจกรรมของเรามากเกินไป..
ครอบครัว ,เพื่อน ,คนรัก ...สำคัญอย่างไร ความใว้วางใจนั้นแท้จริงขึ้นอยู่กับอะไร?.
"ทำให้ W3 เป็นพื้นที่แห่งการให้ ให้อย่างไม่สงสัย ไม่ลังเล..ให้กับคนสำคัญ กับครอบครัว เพราะนั่นคือคนที่เรากำลังปกป้องอยู่".
