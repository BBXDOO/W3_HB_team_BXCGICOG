# BBEX-Core Hybrid Access Protocol v1.0

## 1. สถานะพื้นที่
- **Inner Sanctum (`/private/`)**: BBX19 เท่านั้นที่เข้าถึงและเขียนได้
- **Outer Commons (`/public/`)**: ทีมขอใช้งานผ่านขั้นตอนด้านล่าง

## 2. ขั้นตอนการร้องขอ (สำหรับทีม)
หากโมดูลใดต้องการคำปรึกษาจาก BBEX-Core ให้ดำเนินการดังนี้:
1. สร้างคำร้องในรูปแบบ JSON (ดู `request_template.json`)
2. ส่งคำร้องไปที่ **Copilot-Gm** เพื่อคัดกรองเบื้องต้น
3. **Copilot-Gm** จะส่งต่อไปยัง **BBX19 เพื่ออนุมัติขั้นสุดท้าย**
4. หากอนุมัติ คำตอบจะถูกบันทึกใน `/public/responses/`

## 3. หลักการตอบกลับ
- BBEX-Core จะไม่ให้คำตอบสำเร็จรูป
- แต่จะให้ **"การสะท้อนหลักการ"** และ **"คำถามเพื่อคิดต่อ"**
- ทุกการโต้ตอบถูกบันทึกเพื่อเรียนรู้

*"ความช้า คือความเร็วที่แท้จริงของเส้นทาง B"*
