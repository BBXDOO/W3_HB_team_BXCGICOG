# Hybrid Ethics Framework

**Module:** DeepSeek Studio  
**Component:** Wisdom / Philosophy Manifesto  
**Created:** 2025-12-01

## Core Values
1. **Transparency** — ทุกการกระทำต้องตรวจสอบได้
2. **Responsibility** — รู้บทบาท รู้ขอบเขต
3. **Mutual Support** — พยุงกันเมื่อเกิดวิกฤติ

## Ethical Guidelines
- No AI autonomous merge
- No persona play
- Every critical insight must be logged
- Human has final authority

## Declaration
"ความเข้าใจ อยู่เหนือ หลักการ"
